# Data Labeling Web Service
본 서비스는 2022 한국공학대학교 졸업작품 AI 학습용 데이터 레이블링 시스템입니다. <br>
사용자가 원하는 어떤 데이터든 레이블링 할 수 있는 웹 서비스 제공  <br>
AI를 통한 데이터 레이블링으로 작업 속도 향상 및 비용 감소  <br>
지도학습 AI를 이용한 반자동 레이블링 시스템을 구축하는 것이 목표 <br>
